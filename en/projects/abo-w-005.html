  • Description

    'Book eShop' is one of my first Java Spring Boot web applications. It was created during my studies as a full project for my spring semester. This web application is basically an e-Commerce shop, selling books. Users can sign up, login and buy books that are available in the store. An admin can perform tasks such as adding a new book to the database (in the store), changing details for the books (ex. price) etc.
