  • 'AlexX Book eShop' is my first Java Spring Boot web application. It was created during my studies as a full project for my spring semester of 2020. This web application is basically an e-Commerce shop, selling books. Users can sign up, login and buy books that are available in the store. An admin can perform tasks such as adding a new book to the database (in the store), changing details for the books (ex. price) etc.

  • Technologies used
    • Spring Boot (Java)
    • Thymeleaf Template Engine
    • PostgreSQL Database
