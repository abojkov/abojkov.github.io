  • Description

    Schulthess has been the leading Swiss washing technology group for a long time and stands for innovation in laundry care. As a pioneer, Schulthess constantly breaks new ground with the aim of simplifying everyday life. WashMaster is a digital payment system for your laundry rooms and helps you manage your machine usage in no time. I am currently the main developer of the the terminal application for maintenance and overall improvements. This is a separate application from the main washMaster application, and this app is only used by the building users and admins. It is not published on Google Play Store because only employees from Schulthess can install and distribute this application. The app only works properly when connected with the washing machines with some additional hardware. For additional information, please refer to the links above.
