  • Description

    'StaroNovo' is a platform for selling and buying clothes, either second-hand or manufactured in your own workshop. Everyone can login/register and use this simple and modern web application, and everyone could be either a seller or a buyer. This is the initial release of the application. The web app is still not hosted, but will be in the near future.

  • Technologies used

    * Lumen Micro-Framework By Laravel (PHP)
    * Vue.js Progressive JavaScript Framework
    * MySQL Database
