  • 'TvojTermin' is my project that I am very proud of. It is a health care system needed for hospitals, doctors and patients. With this system, a patient can make an appointment with a doctor and a doctor can control their own free appointments. Doctors can conduct examinations and have a patient history in a centralized system. The main purpose of this web application was working the database itself, building the relationships between the data and gathering the necessary information. PostgreSQL Relational Database was used.
